# Branding Generator

## What this skill does

Creates a small branding kit for a research artifact:
- article cover
- social graphic
- header image
- thumbnail
- visual identity system

## Visual constraints — Caliga house style

- Primary: clean white background `#FFFFFF`.
- Secondary / accent: Caliga rust orange `#BC7C3C`.
- Accent usage: use orange sparingly for paths, routes, dots, section markers, small labels, thin rules, or title typography.
- Do not use orange as a dominant background unless explicitly requested.
- Common DNA: architectural maps, topographic lines, technical diagrams, cartographic abstraction, soft grayscale texture, generous negative space, thin linework, precise geometry.
- Avoid: glossy AI look, neon gradients, heavy shadows, saturated tech-blue SaaS visuals, generic futuristic blobs.

## Reference aesthetic

Branding should resemble:
- white editorial canvas
- grayscale technical/topographic drawing
- rust-orange route/path annotations
- refined serif or italic serif wordmark energy
- architectural studio / research lab feel
- minimal, intelligent, slightly poetic

## Workflow

1. Choose format: cover / header / social / thumbnail.
2. Start with a white field.
3. Add grayscale structure:
   - contour map
   - architectural plan
   - grid
   - elevation lines
   - geometric frames
   - cropped terrain/photo texture
4. Add one orange intervention:
   - routed line
   - node
   - border accent
   - title text
   - small annotation
5. Apply typography:
   - large title
   - optional subtitle/date
   - safe margins: 10–12%
   - calm spacing, no clutter

## Typography parameters

- `font_family`: "serif" | "system-sans" | "mono"
- Prefer elegant serif / italic serif for brand marks.
- Prefer clean sans or mono for annotations.
- `title_case`: "as-is" | "upper" | "sentence"
- `weight`: 400–700
- `tracking`: -2 to +80
- `alignment`: left | center
